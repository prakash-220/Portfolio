## 🚀 QUICK START

### Step 1: Get API Key (FREE)
Visit: https://ai.google.dev/
- Click "Get API Key"
- Create new project
- Copy your key
- Optionally add it to a `.env` file for automated build-time configuration

### Step 2: (Optional) Build‑time Injection
If you're deploying to GitHub Pages or another static host you can inject
the key from your `.env` directly into the pages during your build. Simply
emit a small script containing:
```html
<script>window.GEMINI_API_KEY='YOUR_KEY';</script>
```
and include it before `gemini-chat.js`. That script can be generated by any
tool in your pipeline (shell, Node, etc.) using the environment variable
`GEMINI_API_KEY`. There is no longer any separate helper script required.

### Step 3: Open Chat
Click the avatar in bottom-right corner

### Step 4: Enter Key (if needed)
If the key wasn't provided via config, paste it into the chat form and save

### Step 5: Chat!
Start asking questions about your portfolio
